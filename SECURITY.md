# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.2.x   | Yes       |
| < 1.2   | No        |

## Reporting a Vulnerability

If you discover a security vulnerability in OpenAdeia, please report it responsibly.

**Do not open a public GitHub issue.**

Instead, email **security@openadeia.org** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and aim to provide a fix or mitigation plan within 7 days.

## Scope

This policy covers the OpenAdeia application code, including:

- Backend API (Fastify routes, services, middleware)
- Frontend application (React)
- Database migrations and schema
- Docker configuration and deployment scripts
- CI/CD pipeline

Third-party dependencies are monitored via GitHub Dependabot.
